reNamerX Features

reNamerX is packed with powerful features designed to make batch file renaming fast, easy, and precise. This document covers the main features of the application.

User Interface

Modern Design

  • Clean, intuitive interface with dark and cyberpunk themes
  • Responsive layout that adapts to different window sizes
  • Drag-and-drop support for adding files

File Browser

  • Browse and select individual files or entire folders
  • Filter files by name, extension, or other criteria
  • Sort files by name, date, size, or type

Renaming Operations

Text Operations

  • Replace Text: Find and replace text within filenames
  • Add Text: Add prefixes or suffixes to filenames
  • Remove Text: Remove specific characters or patterns
  • Insert Text: Insert text at specific positions

Case Transformations

  • Convert to uppercase (ALL CAPS)
  • Convert to lowercase (all lowercase)
  • Convert to title case (First Letter Capitalized)
  • Convert to camelCase/PascalCase
  • Convert to snake_case or kebab-case

Numbering

  • Add sequential numbers to filenames
  • Configure start number, increment, and digit padding
  • Place numbers at the beginning, end, or specific position
  • Include leading zeros (001, 002, etc.)

Date and Time

  • Add date/time information to filenames
  • Use file creation or modification dates
  • Use current date/time
  • Configure date/time format

Advanced Features

Regular Expressions

  • Use powerful regex patterns for complex renaming operations
  • Access to capture groups in replacements
  • Full regex support with look-ahead and look-behind assertions

Preview

  • Real-time preview of rename operations
  • Side-by-side view of original and new filenames
  • Highlights potential conflicts or errors

Undo System

  • Multi-level undo/redo support
  • Revert any batch operation
  • History of recent operations

Presets

  • Save frequently used rename operations as presets
  • Apply multiple rules with a single click
  • Export and import presets to share with others

Workflow Enhancements

Keyboard Shortcuts

  • Extensive keyboard shortcuts for power users
  • Customizable key bindings
  • Quick access to common functions

Batch Processing

  • Process hundreds or thousands of files at once
  • Handle large batches efficiently
  • Cancel long-running operations if needed

Cross-Platform

  • Available for Windows (macOS and Linux coming soon)
  • Consistent experience across platforms
  • Native look and feel

System Integration

File System Access

  • Safe file operations that respect permissions
  • Handles various filesystem encodings and special characters
  • Detects and prevents naming conflicts

Extensions

  • Preserve or modify file extensions
  • Batch extension changing
  • Case-sensitive extension handling
