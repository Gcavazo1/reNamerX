Getting Started with reNamerX

This guide will help you install and set up reNamerX on your system, and walk you through your first file renaming operations.

System Requirements

Before installing reNamerX, ensure your system meets the following requirements:

  • Operating System: Windows 10 or Windows 11 (64-bit)
  • RAM: 4GB minimum (8GB recommended)
  • Disk Space: 200MB free disk space
  • Display Resolution: 1280Ã—800 or higher

Installation

Windows Installation

  1. Download the Installer

    • Visit the download page and select the appropriate installer:
    • MSI installer (recommended): reNamerX_1.0.0_x64_en-US.msi
    • Or EXE installer: reNamerX_1.0.0_x64-setup.exe

  2. Run the Installer

    • Double-click the downloaded file to start the installation
    • Follow the prompts in the installation wizard
    • Choose your preferred installation location or use the default

  3. Launch the Application

    • After installation, launch reNamerX from:
    • The desktop shortcut
    • The Start Menu
    • Or by searching for "reNamerX" in the Windows search

Portable Version (No Installation)

If you prefer not to install the application:

  1. Download the portable ZIP file from the download page
  2. Extract the ZIP to a folder of your choice
  3. Run reNamerX.exe directly from the extracted folder

First Launch

When you first launch reNamerX, you'll see the main interface divided into two sections:

  • Left panel: File selection and preview
  • Right panel: Renaming rules configuration

Your First Renaming Operation

Let's perform a simple renaming operation to get familiar with the interface:

1. Select Files

  1. Click the "Browse Files" button or press Ctrl+O
  2. Navigate to a folder with files you want to rename
  3. Select one or more files and click "Open"

Alternatively, click "Browse Folder" or press Ctrl+D to select all files in a directory.

2. Create a Simple Renaming Rule

Let's add a prefix to all selected files:

  1. In the right panel, select "Add Text" from the dropdown
  2. Choose "Add Prefix" option
  3. Enter text you want to add (e.g., "Project_")
  4. Notice the preview updating in real-time in the left panel

3. Apply the Rename

  1. Review the preview to ensure the changes look correct
  2. Click the "Apply" button at the bottom or press Ctrl+Shift+R
  3. Confirm the operation when prompted

Congratulations! You've successfully renamed your first set of files with reNamerX.
