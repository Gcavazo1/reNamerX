Frequently Asked Questions

Get answers to the most common questions about reNamerX.

General Questions

What is reNamerX?

reNamerX is a powerful desktop application for batch renaming files. It offers a modern interface with advanced features for efficiently renaming multiple files using various rules and patterns.

Which operating systems does reNamerX support?

Currently, reNamerX is available for Windows 10 and Windows 11 (64-bit). Support for macOS and Linux is planned for future releases.

Is reNamerX free to use?

Yes, reNamerX is free software released under the MIT License. You can use it for both personal and commercial purposes without cost.

Where can I download reNamerX?

You can download reNamerX from the download page on this website or from the GitHub releases page.

How do I report bugs or request features?

You can report bugs or request features by opening an issue on our GitHub repository. Please check existing issues first to avoid duplicates.

Installation & Updates

Do I need to install any prerequisites?

No, reNamerX is self-contained and doesn't require any additional software to be installed.

Can I use reNamerX without installation?

Yes, we provide a portable version that can be run without installation. Download the portable ZIP file from our download page and extract it to any location.

How do I update reNamerX?

When a new version is released, you can:

  1. Download the latest installer from our website
  2. Run the installer, which will automatically update your existing installation
  3. Your settings and presets will be preserved

Will reNamerX automatically check for updates?

Currently, reNamerX doesn't automatically check for updates. We're planning to add this feature in a future release.

Usage Questions

How many files can reNamerX process at once?

reNamerX can handle hundreds or even thousands of files at once, depending on your system's memory. For very large batches (10,000+ files), we recommend processing them in smaller groups for better performance.

Can I undo renaming operations?

Yes, reNamerX includes a multi-level undo system. You can undo any batch rename operation by pressing Ctrl+Z or clicking the Undo button.

Can I rename files across multiple folders?

Currently, reNamerX processes files within a single folder at a time. Support for processing files across multiple folders simultaneously is planned for a future release.

Does reNamerX support regular expressions?

Yes, reNamerX fully supports regular expressions (regex) for advanced pattern matching and replacement. Enable the regex option in the Replace Text rule to use this feature.

Can I save my frequently used renaming patterns?

Yes, you can save your renaming configurations as presets. Configure your rules, click "Save Preset" or press Ctrl+Alt+S, and give your preset a name. You can later load these presets from the Presets menu.

Can reNamerX rename files based on their metadata?

Yes, reNamerX can rename files based on date/time metadata (creation date, modification date). Support for additional metadata (EXIF, ID3, etc.) is planned for future releases.

Troubleshooting

Why can't I rename some files?

There could be several reasons:

  1. The files might be in use by another application
  2. You might not have permission to modify the files
  3. The target filename already exists
  4. The new filename contains invalid characters

Check the application's status messages for specific error information.

How do I fix "Access Denied" errors?

"Access Denied" errors typically occur when:

  1. The file is open in another application - close the application and try again
  2. You don't have permission to modify the file - run reNamerX as administrator
  3. The file is in a protected system folder - avoid renaming system files

Why does the preview show some filenames in red?

Filenames shown in red in the preview indicate potential conflicts or errors. This typically happens when:

  1. The resulting filename would be invalid
  2. Multiple files would end up with the same name
  3. The new name contains invalid characters

Modify your renaming rules to resolve these issues before applying.

I can't see all my files after adding a folder. Why?

By default, reNamerX shows all files in the selected folder. If you're not seeing some files:

  1. Check if you have an active filter in the search box
  2. Verify that you're looking in the correct folder
  3. Ensure you haven't accidentally enabled a file type filter

Why are my presets missing after updating?

In most cases, presets are preserved when updating reNamerX. If your presets are missing:

  1. Check if you installed to a different location than before
  2. Look in the backup folder (reNamerX creates backups of settings before updates)
  3. If you can't recover them, consider exporting presets to files regularly as a backup

Advanced Features

How do I use capture groups in regex replacements?

When using regex with capture groups:

  1. Enable the "Use regex" option in the Replace Text rule
  2. Create capture groups in your pattern using parentheses: (group1)(group2)
  3. Reference captured groups in your replacement text using $1, $2, etc.

Example: To swap two words, use pattern (\w+)_(\w+) and replacement $2_$1

Can I combine multiple renaming rules?

Yes, reNamerX lets you apply multiple rules in sequence. Each rule takes the output of the previous rule as its input. Add multiple rules in the rules panel and they will be processed in order from top to bottom.

Can I export/import presets between computers?

Yes, you can export presets to files and import them on another computer:

  1. From the Presets menu, select "Manage Presets"
  2. Select the presets you want to export
  3. Click "Export" and choose a location
  4. On the other computer, open the Manage Presets dialog and click "Import"

Is there a command-line interface for reNamerX?

Currently, reNamerX doesn't have a command-line interface. We're considering adding this feature in a future version to support automation and scripting.
