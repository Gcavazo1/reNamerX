reNamerX Usage Guide

This guide covers how to effectively use reNamerX for common file renaming tasks. Learn the interface, renaming techniques, and workflow tips.

Interface Overview

The reNamerX interface is designed to be intuitive and efficient:

Main Sections

  1. Files Panel (Left Side)

    • File selection area
    • File list display
    • Preview of rename operations

  2. Rules Panel (Right Side)

    • Renaming rules configuration
    • Rule type selection
    • Rule parameters

  3. Action Bar (Bottom)

    • Apply button
    • Clear button
    • Status information

Working with Files

Selecting Files

There are multiple ways to add files for renaming:

  1. Browse for Files

    • Click the "Browse Files" button
    • Use keyboard shortcut Ctrl+O
    • Select multiple files using Ctrl or Shift

  2. Browse for Folder

    • Click the "Browse Folder" button
    • Use keyboard shortcut Ctrl+D
    • All files in the selected folder will be added

  3. Drag and Drop

    • Drag files from File Explorer directly into the application
    • Drag folders to add all contained files

Managing Selected Files

  • Select All: Use Ctrl+A to select all files in the list
  • Invert Selection: Use Ctrl+N to invert your current selection
  • Clear Selection: Press Escape to deselect all files
  • Remove Files: Select files and press Delete or click "Remove Selected"
  • Clear All Files: Click "Clear All" or press Ctrl+K

Filtering Files

Use the search box above the file list to filter files by name:

  1. Click in the search box or press Ctrl+/
  2. Type part of a filename or extension
  3. The file list will update to show only matching files
  4. Clear the search to show all files again

Creating Renaming Rules

Basic Text Operations

  1. Replace Text

    • Select "Replace Text" from the rule type dropdown
    • Enter the text to find in "Find What"
    • Enter replacement text in "Replace With"
    • Options: Match case, whole word, use regex

  2. Add Text

    • Select "Add Text" from the rule type dropdown
    • Choose position: prefix, suffix, or specific position
    • Enter the text to add
    • For specific position, specify character position

  3. Remove Text

    • Select "Remove Text" from the rule type dropdown
    • Choose removal type: characters, range, or pattern
    • Specify parameters based on removal type

Case Transformations

  1. Select "Change Case" from the rule type dropdown
  2. Choose the desired case transformation:
    • UPPERCASE
    • lowercase
    • Title Case
    • Sentence case
    • camelCase
    • PascalCase
    • snake_case
    • kebab-case

Sequential Numbering

  1. Select "Add Numbers" from the rule type dropdown
  2. Configure numbering options:
    • Start number (e.g., 1, 001, etc.)
    • Increment (typically 1)
    • Padding (number of digits, e.g., 3 for 001)
    • Position (prefix, suffix, or specific position)
    • Separator (optional character between name and number)

Date and Time

  1. Select "Add Date/Time" from the rule type dropdown
  2. Configure options:
    • Date format (e.g., YYYY-MM-DD)
    • Time format (e.g., HH-MM-SS)
    • Use file dates or current date/time
    • Position in filename

Advanced Techniques

Regular Expressions

For complex renaming patterns, use regular expressions:

  1. Enable "Use regex" in Replace Text rule
  2. Use standard regex syntax for pattern matching
  3. Use capture groups with $1, $2 etc. in replacement

Examples:

  • Match filenames with numbers: .*?(\d+).*
  • Extract text between brackets: \[(.*?)\]
  • Replace with capture group: $1_newtext

Working with Presets

Save frequently used rule combinations as presets:

  1. Saving a Preset

    • Configure your renaming rules
    • Click "Save Preset" or press Ctrl+Alt+S
    • Enter a name for the preset
    • Click "Save"

  2. Loading a Preset

    • Click "Load Preset"
    • Select a preset from the list
    • Click "Load"

  3. Managing Presets

    • Click "Manage Presets"
    • Rename, delete, or export presets

Best Practices

Workflow Optimization

  1. Use Preview Before Applying

    • Always review changes in the preview before applying
    • Switch between list and side-by-side views with Ctrl+P

  2. Start with Small Batches

    • When using complex rules, test with a small batch first
    • Create a test folder with sample files if needed

  3. Use Undo for Safety

    • Remember you can always undo operations with Ctrl+Z
    • Multiple levels of undo are supported

  4. Combine Rules Effectively

    • For complex renaming, combine multiple rules in sequence
    • Consider saving frequent combinations as presets

  5. Filter Before Renaming

    • Use the search feature to work with subsets of files
    • Rename similar files together using filtering

Common Scenarios

Photo organization:

  1. Select "Add Date/Time" with file creation date
  2. Add sequential numbering as suffix
  3. Apply to keep photos organized by date

Document standardization:

  1. Use "Change Case" to convert to Title Case
  2. Replace spaces with underscores
  3. Add department prefix
  4. Apply for consistent file naming

Music library:

  1. Select "Replace Text" with regex enabled
  2. Use pattern like (\d+) - (.*) to capture track numbers and titles
  3. Replace with $1_$2 to reorganize
  4. Apply to standardize music files
