Troubleshooting reNamerX

This guide provides solutions to common issues you might encounter while using reNamerX.

Installation Issues

The installer fails to run

Problem: The installer gives an error or fails to start.

Solutions:

  1. Ensure you have administrative privileges
  2. Try running the installer as administrator (right-click → Run as administrator)
  3. Temporarily disable your antivirus/security software
  4. Download the installer again in case the file was corrupted
  5. Try the portable version instead

Missing dependencies error

Problem: You get an error about missing dependencies.

Solutions:

  1. Install the latest Visual C++ Redistributable for Visual Studio 2019
  2. Make sure your Windows is up to date
  3. Restart your computer after installing dependencies

Startup Problems

Application doesn't start

Problem: When you click the reNamerX icon, nothing happens.

Solutions:

  1. Check if the process is running in Task Manager
  2. Try running the application as administrator
  3. Reinstall the application
  4. Check if your antivirus is blocking the application

Application crashes on startup

Problem: The application starts to load but crashes immediately.

Solutions:

  1. Check if you have enough free disk space
  2. Delete the settings file (located in %APPDATA%\reNamerX) and restart
  3. Try running in compatibility mode for Windows 10
  4. Reinstall the application

File Selection Issues

Can't browse for files

Problem: The file browser dialog doesn't open when clicking "Browse Files".

Solutions:

  1. Try using the keyboard shortcut (Ctrl+O) instead
  2. Restart the application
  3. Check if another dialog is already open
  4. Ensure you have permission to access the file system

Files not showing in the list

Problem: Files aren't appearing in the file list after selection.

Solutions:

  1. Check if you have an active filter in the search box
  2. Try selecting the files again
  3. Ensure the files aren't hidden or system files
  4. Verify you have read permissions for the files

Can't drop files into the application

Problem: Drag and drop doesn't work.

Solutions:

  1. Ensure you're dragging from File Explorer, not another application
  2. Try selecting files using the browse button instead
  3. Restart the application
  4. Run the application as administrator

Renaming Issues

"Access Denied" errors

Problem: You get "Access Denied" errors when trying to rename files.

Solutions:

  1. Close any applications that might be using the files
  2. Run reNamerX as administrator
  3. Check if the files are read-only (right-click → Properties)
  4. Ensure you're not trying to rename system or protected files

Duplicate filename errors

Problem: Renaming fails because of duplicate filenames.

Solutions:

  1. Modify your renaming rules to ensure unique names
  2. Look for filenames in red in the preview, which indicate conflicts
  3. Add sequential numbers to ensure uniqueness
  4. Sort your files differently before applying the rename

Invalid characters in filenames

Problem: Renaming fails due to invalid characters.

Solutions:

  1. Avoid using characters not allowed in filenames: \ / : * ? " < > |
  2. Use the Replace Text rule to remove these characters
  3. Check your regex patterns if you're using regular expressions
  4. Look for filenames in red in the preview, which indicate invalid names

Files not renaming

Problem: Some files aren't being renamed despite being in the list.

Solutions:

  1. Check if the files are locked by another application
  2. Verify you have write permissions for the folder
  3. Ensure the new filenames are valid and don't cause conflicts
  4. Check if the files are selected in the file list

Performance Issues

Slow performance with many files

Problem: The application becomes slow when working with large numbers of files.

Solutions:

  1. Split your renaming tasks into smaller batches
  2. Close other applications to free up memory
  3. Simplify your renaming rules if they're very complex
  4. Avoid using resource-intensive regex operations on large batches

High memory usage

Problem: reNamerX uses a lot of memory.

Solutions:

  1. Process fewer files at once
  2. Restart the application between large batching operations
  3. Close other applications while using reNamerX
  4. Ensure your system meets the recommended requirements

Preview updates slowly

Problem: The preview takes a long time to update when changing rules.

Solutions:

  1. Reduce the number of files in your selection
  2. Simplify your renaming rules
  3. Disable real-time preview temporarily
  4. Use less complex regex patterns

Preferences and Settings Issues

Presets not saving

Problem: Your saved presets disappear after restarting the application.

Solutions:

  1. Ensure you have write permissions for the application data folder
  2. Check if your antivirus is blocking the application from writing files
  3. Export important presets to files as a backup
  4. Try reinstalling the application

Settings not persisting

Problem: Your preferences reset when you restart the application.

Solutions:

  1. Ensure you have write permissions for the application data folder
  2. Check if your settings file is set to read-only
  3. Run the application as administrator once to save settings
  4. Check if your antivirus is blocking writing to the settings file

Recovery and Undo

Undo doesn't work

Problem: The undo function isn't working.

Solutions:

  1. Ensure you haven't performed another operation after the one you want to undo
  2. Check if the files are still accessible in their new location
  3. Restart the application and try again
  4. If possible, manually rename the files back

Need to recover original filenames

Problem: You need to recover original filenames after a mistake.

Solutions:

  1. Use the Undo function (Ctrl+Z)
  2. Check the undo history log (if available)
  3. Restore from a backup if you have one
  4. If you know the pattern of the original names, try to create an inverse renaming rule

Still Having Problems?

If you're still experiencing issues after trying these solutions:

  1. Check the FAQ page for more information
  2. Visit our GitHub Issues page to see if others have reported similar problems
  3. Submit a new issue with:
    • A detailed description of the problem
    • Steps to reproduce the issue
    • Your system information
    • Screenshots if applicable
  4. For urgent help, contact the developer directly through GitHub

Remember to always test your renaming operations on a small batch of files before applying them to your entire collection!
